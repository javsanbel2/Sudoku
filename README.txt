Sudoku validator

Steps to validate sudoku:

1. Rename validate.bat_rename to validate.bat

2. Make validate.bat executable with the following command: 
   2.1 WINDOWS: chmod +x ./validate.bat
   2.2 LINUX or MAC: chmod +x ./validate.bat

3. Run the program using our sudoku.txt or edit this file to have a different result
   3.1 ./validate.bat sudoku.txt


4. Enjoy!
